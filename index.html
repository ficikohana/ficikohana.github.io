Tutorial Membuat Website

Ikuti langkah-langkah berikut untuk membuat website pertama Anda

1

Buat File HTML

Buat struktur HTML dasar dengan DOCTYPE, head, dan body.

<!DOCTYPE html>
<html lang="id">
<head>
  <title>Website Saya</title>
</head>
<body>
  <h1>Hello World!</h1>
</body>
</html>
2

Tambahkan CSS

Styling website dengan CSS atau framework seperti Tailwind CSS.

<link href="https://cdn.tailwindcss.com"
  rel="stylesheet">

<div class="bg-blue-500 text-white
  p-4 rounded">
  Styled Content
</div>
3

Tambahkan JavaScript

Buat website interaktif dengan JavaScript untuk fungsionalitas dinamis.

<script>
  document.getElementById('btn')
    .addEventListener('click', () => {
      alert('Hello World!');
    });
</script>

Upload ke GitHub Pages

Deploy website Anda secara gratis dengan GitHub Pages

1

Buat Repository GitHub

• Login ke akun GitHub Anda

• Klik tombol "New" untuk membuat repository baru

• Beri nama repository: ficikonoha.github.io

• Pastikan repository bersifat Public

• Centang "Add a README file"

• Klik "Create repository"

Tampilan halaman pembuatan repository GitHub

Upload file melalui GitHub web interface

2

Upload File Website

• Masuk ke repository yang sudah dibuat

• Klik "uploading an existing file" atau drag & drop

• Upload semua file website (HTML, CSS, JS, gambar)

• Pastikan file utama bernama index.html

• Tulis commit message, contoh: "Add website files"

• Klik "Commit changes"

3

Aktifkan GitHub Pages

• Masuk ke tab Settings di repository

• Scroll ke bawah hingga menemukan Pages

• Di bagian "Source", pilih Deploy from a branch

• Pilih branch main dan folder / (root)

• Klik Save

• Tunggu beberapa menit untuk deployment

Pengaturan GitHub Pages di repository settings

Website Anda sudah live dan dapat diakses

Website Sudah Live!

• Website Anda akan tersedia di:

https://ficikonoha.github.io

• Proses deployment biasanya memakan waktu 5-10 menit

• Setiap kali Anda update file, website akan otomatis ter-update

• Anda bisa menggunakan custom domain jika diperlukan

Alternatif: Menggunakan Git Command Line

Setup Awal
# Clone repository
git clone https://github.com/ficikonoha/ficikonoha.github.io.git
# Masuk ke folder
cd ficikonoha.github.io
Upload & Deploy
# Tambahkan semua file
git add .
git commit -m "Add website"
git push origin main
# Website akan live dalam 5-10 menit
